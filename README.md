# annual-main-project
Сайт на React

Как запустить:
1) Клонировать данную ветку
2) >> npm i
3) Ожидать установки всех зависимостей
4) >> npm run dev

Deploy версия на Vercel: https://cv-cars-4.vercel.app/

FastApi Api:

1) >> cd ./api
2) >> pip install -r requirements.txt
3) >> uvicorn main:app --reload
